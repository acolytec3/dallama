# Vosk STT Mobile Web App

A mobile-friendly, browser-based speech-to-text (STT) web app using Vosk WebAssembly (WASM), React, Chakra UI, and Vite. The app runs all STT in-browser, supports offline/PWA mode, and provides a modern, accessible UI.

## Tech Stack
- React (TypeScript)
- Chakra UI
- Vite
- vosk-browser (WASM STT)
- Vite PWA plugin

## Setup
```sh
cd vosk-stt-app
npm install
npm run dev
```
